# RioChallenge

## Primeiramente: Criar um arquivo .env na pasta raiz e adicionar sua chave API do google, nesse formato -> `API_KEY="chave"`

## Após isso, seguir os passos abaixo:
### `npm i` Instalar os pacotes utilizados
### `npm run build`
### `npm start`
